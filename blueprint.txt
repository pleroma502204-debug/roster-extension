# 排班小幫手 — 完整檔案對照表

## 根目錄（直接放在專案根目錄）

| 檔案 | 說明 |
|------|------|
| `index.html` | SPA 唯一入口，含三頁 `<template>` |
| `app.css` | 合併後的全域樣式 |
| `app.js` | SPA router（navigate / mount / unmount） |

---

## src/

```
src/
│
├── shared/                        ← 跨模組共享
│   ├── constants.js               ← KEYS、TEMPLATES、DEFAULT_DUTIES 等
│   ├── storage.js                 ← chrome.storage.local 純 I/O（只有 globalState 可以 import）
│   └── utils/
│       ├── date.js                ← parseMonth、daysUntil、formatDate、rocMonthLabel
│       └── dom.js                 ← openModal、closeModal、bindModalClose、showHint、showExportMsg
│
├── core/
│   ├── store/
│   │   └── globalState.js         ← 唯一記憶體真相來源（init / getter / setter / subscribe）
│   └── services/
│       ├── siteService.js         ← recalcLast（純計算）
│       ├── employeeService.js     ← getArrangedCandidates（純計算）
│       ├── holidayService.js      ← buildHolidaySet、buildHolidayDaySet、isHoliday
│       ├── scheduleEngine.js      ← nextLeave、applyClick（純計算）
│       └── exportEngine.js        ← exportCommunityXlsx / exportEmployeeXlsx / exportBigXlsx
│
└── pages/
    ├── data/
    │   ├── main.js                ← 資料頁 mount() / unmount()，管 tab 切換與訂閱
    │   └── components/
    │       ├── siteTab.js         ← 據點列表、据點 Modal CRUD
    │       └── employeeTab.js     ← 人員列表、人員 Modal CRUD、通勤地圖
    │
    ├── schedule/
    │   └── main.js                ← 排班頁 mount() / unmount()，社區班表 + 人員班表
    │
    ├── settings/
    │   ├── main.js                ← 設定頁 mount() / unmount()，基本/假別/勤務/行事曆/合約/缺額
    │   └── components/
    │       └── exportPanel.js     ← 匯出按鈕監聽，呼叫 exportEngine
    │
    └── panels/
        └── main.js
     
```
